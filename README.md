# Moral Explaining Away (GitHub Pages)

A short anonymous experiment about moral judgment and causal discounting ("explaining away") in healthcare AI contexts.

## Deploy
1. Upload these files to a GitHub repo.
2. Enable GitHub Pages (Settings → Pages → Deploy from branch).
3. Set `GOOGLE_SCRIPT_WEBAPP_URL` and `PROLIFIC_COMPLETION_URL` in `js/submit.js`.

## Prolific
Your study URL should include Prolific parameters automatically:
?PROLIFIC_PID=...&STUDY_ID=...&SESSION_ID=...

No completion code is asked at the end; participants are redirected to the completion URL.
